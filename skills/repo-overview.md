# Repo Overview Skill

## Objective

Determine the overall purpose and context of a repository.

Identify what the project does, its domain, and the primary technologies used.

---

# Workflow

## Step 1 — Read Repository Documentation

Check:

- README
- project website
- docs folder
- project description

Extract:

- project purpose
- problem being solved
- main features

---

## Step 2 — Identify Technology Stack

Detect:

- programming languages
- frameworks
- infrastructure components

Look for:

- configuration files
- dependency manifests
- build files

---

## Step 3 — Identify Project Type

Determine whether the project is:

- framework
- application
- library
- infrastructure tool
- AI/ML project

---

# Output Format

Repository Overview

Project Purpose

Technology Stack

Project Domain

Key Features

Primary Documentation

Sources
