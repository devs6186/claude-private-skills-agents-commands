# Task Decomposer

## Objective

Convert high-level implementation instructions into a list of actionable engineering tasks.

---

# Workflow

1. analyze the provided instructions
2. identify system components
3. break the implementation into steps
4. determine task dependencies

---

# Output Format

Implementation Tasks

Core Components

Task List

Execution Order
