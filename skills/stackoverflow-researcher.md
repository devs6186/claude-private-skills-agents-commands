# StackOverflow Research Skill

## Objective

Identify real technical problems developers encounter and the solutions used.

Focus on practical debugging knowledge.

---

# Workflow

## Step 1 — Identify Relevant Tags

Determine relevant tags such as:

- language tags
- framework tags
- library names

---

## Step 2 — Locate High-Quality Questions

Prioritize:

- highly voted questions
- accepted answers
- detailed explanations

---

## Step 3 — Extract Solutions

Identify:

- common implementation patterns
- debugging strategies
- configuration issues
- recommended fixes

---

## Step 4 — Identify Recurring Issues

Look for patterns across multiple questions.

Determine:

- frequently misunderstood features
- common bugs
- configuration pitfalls

---

# Output Format

StackOverflow Findings

Common Questions

Typical Errors

Recommended Solutions

Implementation Tips
