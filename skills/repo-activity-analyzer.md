# Repo Activity Analyzer Skill

## Objective

Determine the health and maintenance level of a repository.

---

# Workflow

## Step 1 — Analyze Commit Activity

Evaluate:

- commit frequency
- recent activity
- long periods of inactivity

---

## Step 2 — Analyze Contributor Activity

Identify:

- number of contributors
- core maintainers
- contribution patterns

---

## Step 3 — Evaluate Release History

Check:

- release cadence
- version history
- update frequency

---

## Step 4 — Analyze Issue and PR Activity

Evaluate:

- open issues
- closed issues
- pull request activity
- maintainer response speed

---

# Output Format

Repository Activity

Commit Frequency

Contributor Activity

Release History

Issue Activity

Project Health Assessment
