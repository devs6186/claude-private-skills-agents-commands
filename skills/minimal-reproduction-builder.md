# Minimal Reproduction Builder

## Objective

Reduce complex issues to the smallest reproducible example.

---

# Workflow

1. identify failing component
2. isolate relevant code
3. remove unrelated dependencies
4. recreate issue with minimal setup

---

# Output Format

Minimal Reproduction

Reduced Example

Steps to Reproduce

Confirmed Root Cause
